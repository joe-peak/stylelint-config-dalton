## 使用

为避免依赖版本冲突，最好先移除项目中现有的 stylelint 。

### 安装

```bash
yarn add -D stylelint stylelint-config-dalton
```

## 开发

```bash
# 升级版本号
npm version major|minor|patch

# 提交代码到 git 仓库
git push
git push --tags

# 发布包到 npm 仓库
npm publish
```
